Project Description:

Problem Statement:

As the president of Nomad Student Organization a campus-based student organization, I can safely say that registering for events at Goucher College is very needlessly complex, time-consuming, and a great disappointment considering the era we live in.

Current Process:

  • Go to OSE → 10min
  • Take form → 1min
  • Fill it out → 15min
  • Scan it → 5min
  • Email it to the advisor → 5min
  • Wait until the advisor signs it and sends it back → 1-5 Days
  • Once you get it back, download and print the document → 10min
  • Walk to OSE and submit it → 20min
  • Wait until they approve it → 3-7 Days

Solution:

Our proposed solution is a web-based platform that will allow student organizations to register events, advisors to sign applications, and OSE to approve or reject applications. The new process will be as follows:

  1. Fill out the form and submit it.
  1. The advisor will receive an email notification and be led to the platform where they can sign the application using a letter format or upload an image of their signature.
  1. OSE will receive an email notification about the application submission.
  1. OSE will review the document and approve it, reject it, or request changes to be made.

Project Application:

Club Use Cases

  1. Create Event: As a club member, I want to create a new event for my club and submit it for approval to the advisor and OSE.
  1. Edit Draft Application: As a club member, I want to edit a draft application for an event that I previously created but have not yet submitted for approval.
  1. Delete Draft Application: As a club member, I want to delete a draft application for an event that I no longer want to submit for approval.
  1. Resend Application: As a club member, I want to resend an application for an event that was previously submitted but has not yet been approved.
  1. Monitor Application Status: As a club member, I want to be able to see the status of an application for an event that was submitted for approval.
  1. Download Application: As a club member, I want to be able to download a copy of an application for an event that was submitted for approval.

OSE Use Cases

  1. Approve Application: As an OSE staff member, I want to be able to approve an application for an event that was submitted by a club.
  1. Reject Application: As an OSE staff member, I want to be able to reject an application for an event that was submitted by a club.
  1. Request Change on the Application: As an OSE staff member, I want to be able to request changes to be made on an application for an event that was submitted by a club.
  1. Download Application: As an OSE staff member, I want to be able to download a copy of an application for an event that was submitted for approval.

Advisor Use Cases

  1. Sign Application: As an advisor, I want to be able to sign an application for an event that was submitted by a club.
  1. Download Application: As an advisor, I want to be able to download a copy of an application for an event that was submitted for approval.
